## 🎯 놀이 규칙 4가지
1. **차례 지키기** 🎯 - 친구가 먼저 놀고 있으면 기다려요
2. **함께 놀기** 🤝 - 혼자 놀지 말고 친구와 함께 놀아요
3. **양보하기** 💝 - 친구에게 먼저 하게 해주어요
4. **도와주기** 🤗 - 어려워하는 친구를 도와주어요
