## 🧮 학습 활동 구성

### 🌟 정직함이 주는 선물 (4가지)
1. **신뢰** 🤝 - 사람들이 나를 믿게 됩니다
2. **성장** 📈 - 실수를 통해 더 나은 사람이 됩니다
3. **평화** 🕊️ - 마음이 편안하고 떴떴합니다
4. **관계** 💝 - 진실한 관계를 맺을 수 있습니다

### 📋 상황별 선택 비교 (4가지)
1. **실수했을 때**: 숙제를 깜빡 잇고 안 했어요
   - 잘못된 선택: 거짓말하기 ❌
   - 올바른 선택: 솔직하게 말하기 ✅

2. **친구와의 관계**: 친구의 물건을 실수로 망가뜨렸어요
   - 잘못된 선택: 모른 척하기 ❌
   - 올바른 선택: 사과하고 책임지기 ✅

3. **시험이나 과제**: 어려운 문제를 혼자 풀 수 없어요
   - 잘못된 선택: 친구 답안 몰래 보기 ❌
   - 올바른 선택: 도움 요청하기 ✅

4. **가정에서**: 용돈을 다른 곳에 썼어요
   - 잘못된 선택: 잃어버렸다고 말하기 ❌
   - 올바른 선택: 사실대로 말하기 ✅
